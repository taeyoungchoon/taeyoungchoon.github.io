* name chage

(cons change age)

* man chage
* cli, list

chage -l

** force a user to change password at the next login

chage -d 0 jack

* syntax

vagrant@precise64:~$ sudo chage -m 0 -M 90 -W 7 -E 2027-1-1 -I 30 -d 2017-1-1 vagrant
