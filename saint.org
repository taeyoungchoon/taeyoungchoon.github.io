SAINT is a commercial vulnerability assessment tool. 
Like Nessus, it used to be free and open source but is now a commercial product. 
Unlike Nexpose, and QualysGuard, SAINT runs on Linux and Mac OS X. 
In fact, SAINT is one of the few scanner vendors that don't support (run on) Windows at all.
